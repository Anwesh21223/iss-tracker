# Goal: THe program aims to pull the coordinates data of the International Space Station(ISS) mainly latitude and longitude as per user's request.


##### Abstract: The program written primarily on python for backend and secondarily with html for the UI utilizes flask which is an open source light_weight web framework build natively on python. Specifically the program exploits the api key feature within the framework to pull real time data requests from 'https://api.wheretheiss.at/v1/satellites/25544' which is supposedly yet another open source project that calculates the coordinates of the ISS using orbital mechanics based on data courtesy of US Space Command (NORAD) via CelesTrak.
